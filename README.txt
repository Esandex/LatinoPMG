
-comando para subir tus commits a github
	git push origin master  

-comando para descargar todos los commits de github a la rama origin/master oculta
	git fetch origin

-comando para combinar tu rama master con origin/master
	git merge origin/master

-comando para agregar todos los cambios y hacer commit a la vez
	git commit -am "AQUI EL NOMBRE DE TU COMMIT "
